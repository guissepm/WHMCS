Module package contains below encoded versions:

- Encoder Version (11.0 + 12.0 + 13.0) PHP (7.4, 8.1,8.2):- This will run on PHP (7.4 & 8.1, 8.2) with Loaders (11.0,12.0, 13.0) in including single package.

Installation:-After unpacking the ZIP file, the files and directories have to be installad on the right locations.

    Place the modules/servers/thesslstorefullv2 directory in the modules/servers/ directory of the WHMCS installation.
    Place the modules/addons/thesslstorefullv2_admin directory in the modules/addons/ directory of the WHMCS installation.
    Place the assets/ssl_resources directory in the assets/ directory of the WHMCS installation.
